3. 展示生成 SQL、动态数据表格和 AI 业务总结。

## 环境提示

- 本次后端测试显式使用 `C:\Java jdk-17\JDK17`。
- 当前系统 `JAVA_HOME` 仍指向 Java 21；IntelliJ 项目 SDK 是 Java 17。命令行运行项目前建议将 `JAVA_HOME` 切换为 Java 17。
- 前端完整引入 Element Plus，因此构建存在单包超过 500 kB 的性能警告，不影响当前演示；后续可通过按需引入和路由懒加载优化。
- Day20 联调创建了少量用户名以 `day20_` 开头的本地测试账号，没有输出或保存其密码和 JWT。
- 分页复测发现并修复了 MyBatis Plus 分页拦截器缺失，同时将 JSqlParser 对齐到 MyBatis Plus 3.5.5 兼容的 4.6 版本。
