## 面试展示建议

1. 先用 README 的架构图说明系统不是普通聊天机器人。
2. 打开架构文档讲解只读数据源、五层 SQL 防线、脱敏和自纠错边界。
3. 打开 Knife4j 按接口指南完成注册、登录、授权和查询。
4. 展示同一问题的 Redis 缓存命中和当前用户查询历史。
5. 使用危险写操作问题展示安全拦截，再说明数据库只读账号是最后一道防线。
